ABOUT

Pengalaman pertama saya dalam membuat portofolio website adalah salah satu momen yang penuh tantangan dan pembelajaran.
Saat itu, saya baru saja memulai perjalanan dalam dunia pengembangan web dan ingin memiliki tempat untuk menampilkan
karya-karya saya kepada dunia.

Saat saya mulai membangun halaman-halaman portofolio, saya merasa terkadang frustrasi karena saya masih harus belajar
banyak hal tentang kode HTML, CSS, dan bahasa pemrograman lainnya. Tetapi, saya terus mencoba, belajar dari tutorial online,
dan mencari solusi atas setiap masalah yang muncul.

Setelah berhari-hari kerja keras, portofolio website saya akhirnya selesai. Saya merasa bangga dengan pencapaian ini dan
senang bisa menampilkan proyek-proyek saya kepada teman-teman, kolega, dan calon klien. Pengalaman pertama ini mengajarkan saya
tentang ketekunan, belajar mandiri, dan pentingnya membagikan karya-karya kita dengan dunia.
